# Weather Data Analysis Project

# Project Overview
This project focuses on analyzing weather/climate data using Python. The dataset (climate_data.csv) is processed to understand temperature, humidity, rainfall, and monthly weather trends using data visualization techniques.

# Objective

1. To analyze climate data
2. To identify temperature, humidity, and rainfall patterns
3. To visualize data using graphs
4. To extract meaningful insights from weather data
   
# Tools Used

Python 
Pandas
NumPy
Matplotlib
Jupyter Notebook / Google Colab
